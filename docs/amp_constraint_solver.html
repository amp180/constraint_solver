amp_constraint_solver package¶

Module contents¶

Constraint solving using domain reduction.

Imports * from amp_constraint_solver.constraint_solver and amp_constraint_solver.builtin_constraints.

Usage of amp_constraint_solver.constraint_solver.solve():

>>> from amp_constraint_solver import solve
>>> def constraint1(b, **variables):
...     return variables['a'] != 5 and b != 3
...
>>> constraint2 = lambda a, b: a != b
>>> for solution in solve({'a': [1, 5], 'b': [1, 2, 3]}, [constraint1, constraint2]):
...    print(solution)
...
{'a': 1, 'b': 2}

amp_constraint_solver.constraint_solver module¶

Contains implementation of a domain reduction constraint resolver.

Usage of solve():

>>> from amp_constraint_solver import solve
>>> def constraint1(b, **variables):
...     return variables['a'] != 5 and b != 3
...
>>> constraint2 = lambda a, b: a == b
>>> for solution in solve({'a': [1, 5], 'b': [1, 2, 3]}, [constraint1, constraint2]):
...    print(solution)
...
{'a': 1, 'b': 1}

Glossary:

  • variables: Things that are assigned values in a possible solution.
  • values: Possible values for variables in a solution.
  • domains: Bags of possible valid values for a variable.
  • constraints: Conditions that are used to check if solutions are valid.

class amp_constraint_solver.constraint_solver.SolutionsType¶

Type annotation for a dictionary of variable names mapped to values (a candidate solution.)

class amp_constraint_solver.constraint_solver.Constraint¶

Type annotation for a function that takes in parameters that share their name and type with variables being solved for (or takes kwargs) and returns bool. Used to check if a solution is valid. Should return true if valid arguments are passed, otherwise false.

class amp_constraint_solver.constraint_solver.SolutionGenerator¶

The type returned by solve, It’s a generator of dicts that map variables to values (a generator of SolutionType.).

A recursive generator function that yields solutions to a constraint solving problem, using domain reduction.

eg.

>>> from amp_constraint_solver import solve
>>> list(solve({'a': [1, 2], 'b': [1, 2]}, [(lambda a, b: a > b),]))
[{'a': 2, 'b': 1}]
Parameters:
  • domains (Dict[str, List[~ValueType]]) – A dict of variable names to lists of possible assignments, DomainsType.
  • constraints (List[Callable[…, bool]]) – A list of Constraint functions to check possible solutions.
  • sorted_function – Can be used to override what order variables are assigned in.
Return type:

Generator[Dict[str, ~ValueType], None, None]
Returns:

A generator of candidate solutions. SolutionGenerator
